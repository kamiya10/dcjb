# 狄西機霸隱私權暨個人資料保護政策

狄西機霸（以下稱「本服務」）重視您的隱私權。為了讓您更加瞭解當您使用本服務時，我們在蒐集、處理、利用您個人資料方面的做法以及您依法應享有的權利等事宜，我們提供以下說明。請您仔細閱讀以下內容，若您不同意我們的做法，請暫時不要使用本服務。

## 一、本隱私權政策適用範圍

本隱私權政策適用於本服務網站及其相關之個人資料蒐集、處理及利用事宜。本隱私權政策不適用於本服務以外的網站，包含您透過本服務連結至的第三方網站。

## 二、個人資料蒐集應告知事項

1. **蒐集主體**：狄西機霸

2. **蒐集之目的**：
   - 會員管理與服務
   - Discord 機器人平台之營運
   - 客戶服務及申訴處理
   - 其他經營合於營業登記項目或組織章程所定之業務

3. **個人資料之類別**：
   - Discord 用戶資訊（用戶 ID、用戶名稱、頭像）
   - 電子郵件地址（若 Discord 帳號有提供）
   - 您在本服務上的活動紀錄
   - 您所上架的機器人相關資訊

4. **個人資料利用期間**：
   - 會員帳號存續期間
   - 依相關法令規定或契約約定之保存年限
   - 因執行業務所必須之保存期間

5. **個人資料利用地區**：本服務伺服器所在地及其他與本服務具有正當合作關係之機構營業處所所在地

6. **個人資料利用對象**：本服務及其技術服務提供者

7. **個人資料利用方式**：依本隱私權政策及相關服務條款之規定進行利用

## 三、Discord OAuth2 登入機制說明

1. 本服務使用 Discord OAuth2 進行用戶認證，這意味著：
   - 我們不會儲存您的 Discord 密碼
   - 我們不會儲存 Discord 的 Access Token
   - 登入時僅會取得必要的基本資訊（用戶ID、用戶名稱、頭像）

2. 資料安全性：
   - 所有的認證過程都經過加密傳輸
   - 我們採用業界標準的安全措施保護您的資料
   - 定期進行安全性檢查和更新

## 四、個人資料的修改與刪除

1. 您可以隨時登入本服務檢視和更新您的個人資料
2. 您可以要求刪除您的帳號，但部分資料可能因法律要求而保留一段時間
3. 您的機器人相關資訊將在您刪除時一併移除

## 五、Cookie 的使用

本服務使用 Cookie 及相關技術以提供更好的服務體驗。我們使用 Cookie 來：
1. 維持您的登入狀態
2. 記住您的使用偏好
3. 分析網站流量和使用情況

您可以透過瀏覽器設定來管理 Cookie，但這可能影響部分服務功能的使用。

## 六、資料安全保護措施

1. 所有資料傳輸均使用 HTTPS 加密
2. 定期進行安全性檢查
3. 限制內部人員接觸用戶資料的權限
4. 定期備份資料以預防資料遺失

## 七、未成年人的保護

若您未滿20歲，請在使用本服務前取得法定代理人的同意。

## 八、您的權利

依據個人資料保護法，您就您的個人資料享有以下權利：
1. 查詢或請求閱覽
2. 請求製給複製本
3. 請求補充或更正
4. 請求停止蒐集、處理或利用
5. 請求刪除

## 九、隱私權政策的修改

本服務保留隨時修改本隱私權政策的權利，修改時將於網站公告。建議您定期查看本政策以獲得最新資訊。

## 十、與我們聯絡

如果您對本隱私權政策有任何疑問，或想行使您的相關權利，請透過本服務網站的聯絡方式與我們聯繫。
